# AI Productivity Tools - VS Code Extension

A powerful VS Code extension that brings AI-powered productivity tools directly into your editor.

## Features

- **Writing Assistant**: Improve emails, blog posts, and general text
- **Code Generator**: Generate code in multiple programming languages
- **AI Chat**: Ask anything and get instant responses

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "AI Productivity Tools"
4. Click Install

## Configuration

After installation, configure your API keys:

1. Go to VS Code Settings (File > Preferences > Settings)
2. Search for "AI Productivity Tools"
3. Enter your OpenAI API Key in the `apiKey` field

## Getting an API Key

1. Visit [OpenAI Platform](https://platform.openai.com)
2. Create an account or sign in
3. Go to API Keys section
4. Create a new secret key
5. Copy and paste into VS Code settings

## Usage

1. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
2. Type "AI Productivity Tools: Open Panel"
3. Use any of the three tools:
   - **Writing Assistant**: Enter text and select improvement type
   - **Code Generator**: Describe what code you need
   - **AI Chat**: Ask any question

## Commands

- `AI Productivity Tools: Open Panel` - Opens the AI tools panel

## Requirements

- VS Code 1.80.0 or higher
- OpenAI API key (for AI features)
